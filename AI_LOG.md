# AI Log – Mini E-handel

## Prompt 1
**Fråga:** Hjälp mig strukturera README för mini e-handel med Use Case och kravlista.  
**Resultat:** Fick förslag på struktur, rubriker och möjliga punkter att ta med.  
**Vad jag gjorde:** Använde som utgångspunkt och skrev egen text med exakta funktionella krav, prioritering och Use Case.  

## Prompt 2
**Fråga:** Föreslå aktivitetsdiagram för “lägg i varukorg” baserat på flöde i HTML/JS.  
**Resultat:** Fick ett textbaserat flöde med beslutspunkter.  
**Vad jag gjorde:** Anpassade flödet till min implementation med cart-count update, checkout-beslut och visuell feedback.  

## Prompt 3
**Fråga:** Ge exempel på UML-klassdiagram för Product, Cart, UserInterface och Category.  
**Resultat:** Fick klasser, attribut och relationer.  
**Vad jag gjorde:** Lagt till egna metoder, enum-värden och relationer som matchar min kod och UI.
